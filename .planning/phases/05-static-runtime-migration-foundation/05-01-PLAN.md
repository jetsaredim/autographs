<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Oracle/Object Storage source to public artifacts | Private identifiers and unpublished records must be stripped before static output exists. |
| generated release to anonymous browser | Public HTML/JSON/media paths are directly downloadable after cutover. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Information Disclosure | generated JSON/HTML/manifests | mitigate | Add deny-list scans for `storageNamespace`, `bucketName`, `objectKey`, `objectstorage`, `OCI_`, private image UUID patterns, and original filenames. |
| T-05-02 | Tampering | public contract versioning | mitigate | Include `schemaVersion` in generated catalog/facet/detail/manifest artifacts and test version presence. |
| T-05-03 | Denial of Service | oversized fixture artifacts | mitigate | Profile single-file, split-file, and hybrid outputs against 500 fixture items before freezing the first contract. |
| T-05-SC | Tampering | cargo dependencies | mitigate | Use only crates required for serialization/testing; commit lockfile changes and avoid unvetted generators. |
</threat_model>
