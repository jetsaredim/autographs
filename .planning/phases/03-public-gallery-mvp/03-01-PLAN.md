## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Oracle catalog data to public UI props | Server-safe `AutographItem` objects include storage metadata that must be stripped before public Client Components and DOM output. |
| Quote research to public runtime | Candidate quotes are untrusted until user-approved and copied into the durable approved runtime list. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01-01 | Information Disclosure | `public-view-models.ts` | mitigate | Strip `storageNamespace`, `bucketName`, `objectKey`, `checksum`, and `etag`; verify with grep against public view-model/component files. |
| T-03-01-02 | Tampering | `EmptyState.tsx` | mitigate | Render quote and metadata strings as React text; do not use `dangerouslySetInnerHTML`. |
| T-03-01-03 | Repudiation | quote workflow | mitigate | Preserve `03-QUOTE-CANDIDATES.md` with approval record before runtime quotes are created. |
| T-03-01-04 | Information Disclosure | image path helper | mitigate | Generate only `/api/catalog/{itemId}/images/{imageId}` paths and never direct Object Storage or signed URLs. |
</threat_model>
