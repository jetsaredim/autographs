|----------|-------------|
| Query string to Oracle-backed list | Anonymous users control `signer`, `category`, and `tag`; repository bind variables must remain the filter path. |
| Server catalog records to browser grid | Published items must be shaped into public view models before grid/filter Client Components receive data. |
| Browser image display to private media route | Inline images request app-mediated image routes only. |
